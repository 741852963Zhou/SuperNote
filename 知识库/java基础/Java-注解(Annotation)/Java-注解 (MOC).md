
Java 注解 (Annotation) 是一种==元数据 (Metadata)==，它为代码提供附加信息，但**不直接影响代码的执行**。注解可以被编译器、工具或运行时库（通过反射）读取和处理。

它们在现代 Java 框架（如 Spring）中扮演着至关重要的角色，用于==配置==、==代码生成==、==编译时检查==等。
